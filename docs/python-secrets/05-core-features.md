# Core Features

## Primary Content Types

### 1. Courses (Formations)
- **Format**: Course content + integrated challenges
- **Structure**: Chapters and lessons
- **Assessment**: LeetCode-style challenges OR quizzes within each lesson

### 2. Challenges
- **LeetCode-style**: Coding problems with automated test cases
- **Quiz-style**: Multiple choice questions for concept verification
- **Integrated**: Challenges embedded within courses as practice

### 3. Projects
- **Real-world applications**: Build complete applications
- **Portfolio-worthy**: Projects users can showcase
- **Complexity**: Multi-file projects with full setup

## Design Philosophy

**Keep it simple initially**
- Focus on core learning experience
- Minimize distractions
- Prioritize features that drive conversions
- Add gamification/social only after proving revenue model

## Revenue-Focused Features

Priority order for feature development:
1. **Content delivery**: Course consumption experience
2. **Payment flow**: Seamless purchase process
3. **Skill assessment**: Challenges that demonstrate value
4. **Progress visibility**: Show learning outcomes
5. **Portfolio building**: Projects that prove skills
