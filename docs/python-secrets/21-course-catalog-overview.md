# Course Catalog Overview

## Platform Vision

**Python Secrets** is a Python-only learning platform focused on taking beginners from zero to advanced software engineering expertise in Python and backend development.

### Target Audience
- **Primary**: Beginners who want to learn Python (not complete beginners - they have basic computer literacy)
- **Secondary**: Developers from other languages wanting to learn Python efficiently
- **Tertiary**: Intermediate Python developers wanting to advance to senior/backend roles

### Positioning vs. Other Platform
- **Python Secrets**: Python-only, deep expertise, advanced SE patterns
- **Other Platform**: Multi-language, broader focus

---

## Course Structure

### Learning Philosophy
- **70% practice, 30% theory** - Learn by doing
- **Short, focused lessons** (5-15 minutes each)
- **Immediate feedback** with interactive code editor
- **Real-world projects** - not toy problems
- **Progressive difficulty** - fundamentals → advanced → mastery
